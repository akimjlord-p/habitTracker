# habiTTracker

Учебный проект на rust: Cli приложение для отслеживания работы над привычками.

## Цель проекта

Проект создавался для отработки взаимодействия с git; отработки Options, Result, HashMap и Vector в rust; изучения работы с json на rust; изучения docker;
Ключевые задачи:
- Изучить работу Docker
- Отработать Options, Result
- Изучить работу с json
- Отработать ведение проекта на github

## Используемые технологии

Rust; хранение данных в json на serde
